Contributing and Reporting Bugs

If you would like to contribute financial support to the project, the best way to proceed is to purchase a license for one of theWhiteboxTools Extension productsor by purchasing aWhiteboxTools Open-Core binary. The proceeds from the sales go towards paying developers who work on the open-core portion of the WhiteboxTools platform. Each license sale goes a long way towards making the ongoing development of WhiteboxTools more sustainable.

If you would like to contribute to the project as a developer, follow these instructions to get started:

Fork theWhiteboxToolsproject ( https://github.com/jblindsay/whitebox-tools )Create your feature branch (git checkout -b my-new-feature)Commit your changes (git commit -am 'Add some feature')Push to the branch (git push origin my-new-feature)Create a new Pull Request
